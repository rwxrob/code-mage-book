== Ward Glyph (Error Handling)
*Purpose:* Teach try/catch blocks.

*Spell:* 
`ward glyph <action> from <error>`

*Example:*
[source, go]
----
file, err := os.Open("spellbook.txt")
if err != nil {
    fmt.Println("Error:", err)
    return
}
defer file.Close()
fmt.Println("Successfully opened the spellbook!")
----
