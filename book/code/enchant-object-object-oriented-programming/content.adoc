== Enchant Object (Object-Oriented Programming)
*Purpose:* Teach encapsulation and methods.

*Spell:* 
`enchant <object> with <method>`

*Example:*
[source, go]
----
type Wand struct{}

func (w Wand) ShootFireball() {
    fmt.Println("Shooting fireball!")
}

wand := Wand{}
wand.ShootFireball()
----
